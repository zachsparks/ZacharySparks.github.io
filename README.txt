Projects
Admin Dashboard: Created with React
Description: This project is an admin dashboard built using React, Material UI, Nivo Charts, Formik, Yup, FullCalendar, and Data Grid. It offers both Light and Dark Modes, four different Charts, three Data Table Pages, an FAQ Page, a Form Page, and Calendar Integration.

Forecasthub: OpenWeatherMap API
Description: Forecasthub is a web-based weather application that empowers users to access real-time weather data and forecasts for specific locations worldwide. It harnesses the OpenWeatherMap API, delivering up-to-the-minute weather conditions, providing valuable insights for making informed decisions regarding destinations and activities.

CheckIt: Native JavaScript
Description: CheckIt is a To-Do List web application offering comprehensive features for task creation, management, and filtering. Users can easily add tasks, mark them as completed, and remove them from the list. The application ensures data persistence through local storage, allowing users to pick up where they left off.
